# web_project

*Using nodejs reactjs and mongodb*

## setup instructions

backend files are in the backend folder and frontend files are in the frontend folder. you have to go into the folders seperately and install the required packages in respective folders.

### For backend
go into the backend folder - $ cd backend

install packages - $ yarn

start the server - $ yarn start



### For frontend
go into the frontend folder - $ cd frontend

install packages - $ yarn

start the server - $ yarn start
